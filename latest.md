### 新增功能
1. **直接打开日志** - 支持直接打开日志文件功能，方便查看和管理日志
2. **Split 上下调节** - 支持 split 面板上下拖动调节，优化发送栏显示效果
3. **安装路径选择** - 构建的安装包支持用户选择自定义安装路径
4. **一键发布脚本** - 添加一键发布脚本，发布时自动添加发布描述

### 优化修复
1. **UI 全面优化** - 优化控制栏、间距排布、固定滚屏按钮、清空/连接按钮样式、串口设置显示、命令运行显示、rx/tx 显示等多项界面细节
2. **终端体验优化** - 终端滚动条仅在鼠标悬停时显示，固定滚屏提示默认开启，串口设置被压缩时自动换行显示
3. **设置页面优化** - 设置里配置项小标题做区分，删除多余描述，统一文本按钮样式
4. **日志优化** - 日志分片大小默认值修改为 20M，修改打开日志为打开日志所在文件夹，日志输出均用英文
5. **系统信息优化** - CPU、内存显示策略修改为使用系统数据
6. **菜单栏增强** - 设置入口也加到顶部菜单栏里，方便快速访问
7. **Bug 修复** - 修复 COM 侧边栏断开后选项卡未同步断开、新增组回车弹出重复提示、命令组发送命令无效等问题
8. **CI/CD 优化** - 优化 GitHub Actions 构建逻辑
