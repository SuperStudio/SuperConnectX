### 新增功能
1. **分屏显示功能** - 支持拖拽分屏，可将终端区域拆分为多窗口同时查看不同连接内容，提升多任务操作效率
2. **拖拽排序功能** - 支持选项卡拖拽排序和命令列表拖拽排序，方便自由调整连接和命令的显示顺序
3. **深浅模式支持** - 支持浅色/深色主题切换，适配不同使用环境和视觉偏好
4. **命令编辑优化** - 新增命令改为直接在表格内插入编辑，无需弹出对话框，操作更流畅；自动换行功能添加快捷键入口

### 优化修复
1. **UI 全面优化** - 颜色统一规范化，托盘图标变清晰、右键菜单样式优化，高亮下拉列表选中颜色更明显，侧边栏卡片悬浮不再浮动，tooltip 交互逻辑优化
2. **消息提示改进** - el-message 组件国际化适配并精简冗余消息，ElMessage 改为半透明可穿透点击，设置菜单弹出点击外部区域可正常关闭
3. **连接状态修复** - 修复串口关闭选项卡后左侧串口仍显示连接中的状态不同步问题，未连接状态下命令输入保持使能可用
4. **架构重构** - app.vue 和 ipcConnector 模块重构，优化代码结构，提升可维护性
5. **其他修复** - 修复任务栏图标显示问题、编码及测试报错，语法高亮仅在修改时才保存，补全测试用例，更新多语言 README
