### 新增功能
1. **FTP 服务端与客户端** - 支持 FTP 服务端和客户端连接功能，实现 FTP 配置保存和编辑修改
2. **连接类型重构** - 重构连接类型架构，新建连接改为 Tab 方式，将新建连接组件独立，优化新建连接体验
3. **密码加密显示** - 支持密码加密显示功能，保护敏感信息
4. **测试体系搭建** - 添加 vitest 测试框架，补全大量单元测试和集成测试用例，构建时自动跑测试
5. **Enter 发送命令** - 修改为 Enter 键发送命令，提升操作便捷性
6. **多语言覆盖增强** - 扩展多语言覆盖范围，完善中英文翻译

### 优化修复
1. **UI 美化** - 数字调节按钮美化、统一 tooltip 样式、语法高亮下拉列表 UI 优化
2. **FTP 连接修复** - 修复若干 FTP 连接问题，提升 FTP 连接稳定性
3. **删除连接修复** - 修复删除连接失败的问题
4. **Type Check 修复** - 修复 type check 编译问题
5. **自动归档优化** - 手动触发时不再自动归档，避免不必要的操作
6. **构建流程优化** - 构建时自动运行测试用例，保障代码质量
