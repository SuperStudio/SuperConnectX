### 新增功能
1. **Linux ANSI 颜色支持** - 终端支持 Linux ANSI 颜色转义序列，兼容 Linux 终端的颜色输出显示
2. **语法高亮右键拷贝** - 语法高亮规则支持右键拷贝功能，方便快速复制和编辑规则
3. **语法高亮默认规则** - 语法高亮提供默认可用的高亮规则，开箱即用
4. **官网下载按钮** - 顶部菜单栏添加官网下载按钮，方便快速访问官网获取最新版本
5. **更新对话框增强** - 更新对话框增加更多交互信息，提升更新体验

### 优化修复
1. **检查更新修复** - 修复检查更新功能失败的问题，确保自动更新流程正常运行
2. **菜单栏 UI 修复** - 修复菜单栏弹出时撑开界面布局的问题
3. **字体菜单优化** - 字体菜单栏移动到最底部，避免遮挡其他界面元素
4. **语法高亮编辑修复** - 修复点击编辑语法高亮时未切换到当前设置的高亮规则的问题
5. **正则表达式修复** - 修复正则表达式删除后实时预览未生效的问题；修复正则表达式输入 `|` 字符导致卡死的问题
